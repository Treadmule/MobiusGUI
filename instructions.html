Instructions:

This web application is in it's uttermost early stage... That means, if something doesn't work it's definitely my fault.

At the moment, only firmware version 1.13 is supported.

This app works best with Google Chrome, internet explorer is not tested and therefore (probably) unsupported and I don't intend to support it in the future. Apple's Safari works just fine, but doesn't allow to automatically download the generated file, so you will simply have to hit ⌘+S to save it and name it "SYSCFG.TXT"

If you find any bug please report it here.
